ForgeFetchAPI (Original by MrNavaStar)

Version 1.3 - Created by MrNavaStar (updated by BussyBakks)

This api provides a simple way to get information about minecraft forge
installer versions. This information is provided via the versions.json file.

This versions.json will be updated daily (thanks to github action lmao :=]).

This original source code can be found on GitHub (improvement on this git).
